Part 1: The Basics (What You Need to Know)

1.1 What is OpenClaw?

Simple explanation: OpenClaw is software that lets you create "AI agents" ‚Äî programs that can do tasks for you automatically using artificial intelligence.

Even simpler: You know how ChatGPT can write emails or answer questions? OpenClaw lets you create a version of ChatGPT that works on its own, 24/7, doing specific tasks you assign.

Example: You could create an agent that:

  • Checks your email every hour
  • Drafts responses to simple questions
  • Sends you a summary of what it did
  • Never sleeps, never takes a break

1.2 What is a Terminal (And Why Do I Need It)?

A terminal (also called "command line" or "command prompt") is a way to control your computer by typing commands instead of clicking buttons.

Think of it like this: When you use your computer normally, you're using the "point and click" method. The terminal is the "type commands" method. Some software (like OpenClaw) is installed and run through the terminal.

1 How to Open Terminal on Mac

Method 1 (Easiest):

  1. Press Command + Space on your keyboard (hold Command, press Space)
  2. A search box appears at the center of your screen
  3. Type: terminal
  4. Click the "Terminal" app that appears (black square icon)

Method 2:

  1. Open Finder (the blue smiling face in your dock)
  2. Click "Applications" in the left sidebar
  3. Scroll down and open the "Utilities" folder
  4. Double-click "Terminal"

What you should see: A black window with white text. At the top, it says something like Last login:... and at the bottom there's a line ending with a $ symbol.

2 How to Open Terminal on Windows

Method 1 (Easiest):

  1. Press the Windows key on your keyboard (or click the Start button)
  2. Type: cmd
  3. Click "Command Prompt" or press Enter

Method 2:

  1. Right-click the Start button (Windows icon)
  2. Click "Windows Terminal" or "Command Prompt"

What you should see: A black window with white text showing something like C:\Users\YourName>

üí° Pro Tip: Bookmark this! You'll be using the terminal a lot with OpenClaw. Consider dragging the Terminal/Command Prompt icon to your dock/taskbar for easy access.

1.3 What is an API Key (And Why Do I Need One)?

An API key is like a password that lets OpenClaw use AI services (like ChatGPT, Claude, or others).

Think of it like this: When you log into a website, you use a username and password. When OpenClaw wants to use an AI service, it uses an API key. It's proof that you're allowed to use the service.

Why you need multiple API keys:

  • Different AI companies offer different capabilities
  • Some are better at certain tasks than others
  • Some are cheaper than others
  • Having multiple options gives you flexibility
‚ö†Ô∏è SECURITY WARNING: API keys are like passwords. Never share them, never post them online, never email them to anyone. If someone gets your API key, they can use AI services and charge your account. We'll show you exactly how to keep them safe.

Part 3: Installing Node.js

OpenClaw requires Node.js to run. Think of Node.js as the engine that OpenClaw runs on. You need to install it before you can install OpenClaw.

3.1 Check If You Already Have Node.js

1 Open Your Terminal

Remember Part 1? Open your terminal now:

  • Mac: Press Cmd+Space, type "terminal", press Enter
  • Windows: Press Windows key, type "cmd", press Enter

2 Check Node Version

Type this command exactly (then press Enter):

node --version

If you see something like: v18.17.0 or higher ‚Üí You have Node.js! Skip to Part 4.

If you see: "command not found" or an error ‚Üí You need to install Node.js. Continue below.

3.2 Install Node.js on Mac

1 Download the Installer

  1. Go to: nodejs.org
  2. Click the big green button that says "LTS" (Long Term Support)
  3. Download the macOS Installer (.pkg file)
  4. Wait for the download to complete (it's about 70MB)

2 Run the Installer

  1. Open your Downloads folder
  2. Double-click the node-v... file you just downloaded
  3. Click "Continue" on each screen
  4. Click "Agree" to the license agreement
  5. Click "Install" (you may need to enter your Mac password)
  6. Wait for installation to complete
  7. Click "Close"

3 Verify Installation

  1. Open Terminal (Cmd+Space, type "terminal", Enter)
  2. Type: node --version
  3. Press Enter

You should see a version number like v20.10.0. If you do, Node.js is installed!

3.3 Install Node.js on Windows

1 Download the Installer

  1. Go to: nodejs.org
  2. Click the big green button that says "LTS"
  3. Download the Windows Installer (.msi file)
  4. Wait for download to complete

2 Run the Installer

  1. Open your Downloads folder
  2. Double-click the node-v... file
  3. Click "Next" on each screen
  4. Accept the license agreement
  5. Click "Install" (you may see a User Account Control prompt ‚Äî click "Yes")
  6. Wait for installation
  7. Click "Finish"

3 Verify Installation

  1. Open Command Prompt (Windows key, type "cmd", Enter)
  2. Type: node --version
  3. Press Enter

You should see a version number. If you do, Node.js is installed!
