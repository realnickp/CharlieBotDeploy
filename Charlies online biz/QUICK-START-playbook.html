Part 3: Don't Go Broke (15 minutes)

Token Costs (Simplified)

API calls cost money. Here's how to not panic when you see the bill:

  • GPT-4: Expensive but smart. Use for final output.
  • GPT-3.5: Cheap and fast. Use for sorting, filtering, simple tasks.
  • Rule of thumb: If it's just organizing data, use GPT-3.5. If it's writing content for humans, use GPT-4.

Set a Daily Limit

# In your config.json, add: { "rate_limits": { "max_daily_cost": 5.00 } } # This stops everything if you hit $5/day # Safety net so you don't wake up to a $200 bill
